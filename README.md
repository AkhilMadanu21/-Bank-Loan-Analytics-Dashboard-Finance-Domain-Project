## 📌 Project Overview

This project is an end-to-end **Finance Domain Analytics Dashboard** developed using **Microsoft Power BI**. The dashboard provides comprehensive insights into bank loan operations by analyzing loan applications, funding, repayments, customer risk, and lending performance.

The objective of this project is to help financial institutions monitor loan portfolios, evaluate lending performance, identify credit risks, and support data-driven business decisions.

---

## 🎯 Business Problem

Banks process thousands of loan applications every year. Monitoring loan performance, customer repayment behavior, and loan distribution manually is difficult.

This dashboard helps decision-makers to:

- Monitor overall lending performance
- Analyze good loans vs bad loans
- Track monthly loan trends
- Evaluate customer borrowing behavior
- Identify high-risk loan segments
- Improve strategic lending decisions
